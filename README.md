### What is this project?

This project is a [__Qt6__](https://www.qt.io/development/qt-framework/qt6) based file manager meant for
[_TWM_](https://en.wikipedia.org/wiki/Tiling_window_manager) users. Many of us, being tired of heavy,
[_DE_](https://en.wikipedia.org/wiki/Desktop_environment)-based file managers like Dolphin (KDE),
Nautilus (GNOME)... with unapologetically many dependencies, want something light. While light and utility
focused file managers exist, like PCManFM, the visuals and UI are often unpolished and dated.
[___QtwmFM___](https://github.com/Siddhartha351/QtwmFM) tries to fix that.

> [!NOTE]
> This project is a work in progress.
> So, usual repository features like: documentation, releases, binaries, etc., might not be available.
